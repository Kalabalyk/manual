    Отображение группы распределения в списке и привязка к зоне

    1. Отображение группы распределения в списке

    1. Откройте "Allocation List" (Список распределения) в интерфейсе XPS
    2. Найдите нужную Allocation Group (Группу распределения) в списке
    3. Для удобства используйте:
      • Фильтрацию по названию группы
      • Сортировку по типу контейнеров
    4. Выбранная группа будет выделена в списке

    2. Привязка группы к выбранной зоне терминала

    1. В режиме "Yard View" (План терминала):
      • Выберите инструмент "Select Space" (Выбор зоны)
      • Обведите нужную область на плане (прямоугольник/полигон)
    2. Из контекстного меню:
      • Выберите "Link to Allocation Group" (Привязать к группе)
      • Укажите нужную группу из списка
    3. Подтвердите привязку:
      • Нажмите "Apply" (Применить)
      • Убедитесь, что зона изменила цвет/маркировку

     

    3. Проверка корректности

    • В "Allocation List":
      • У привязанной группы появится пометка о связанной зоне
      • Можно просмотреть все связанные зоны для группы
    • В "Yard View":
      • Связанные зоны отображаются специальным цветом/шаблоном

    Примечание: Для сложных конфигураций можно привязывать несколько зон к одной группе или создавать перекрывающиеся зоны с разными приоритетами.
