  1. Фильтры распределения (Allocation Filters)

    Фильтры распределения для 20' и 40' контейнеров

    1. По заполненности  

    • 20FULL - 20' полные
    • 20EMPTY - 20' пустые
    • 40FULL - 40' полные
    • 40EMPTY - 40' пустые

    2. По опасности груза

    • 20HAZ - 20' опасные грузы (DG)
    • 20NON-HAZ - 20' неопасные
    • 40HAZ - 40' опасные грузы (DG)
    • 40NON-HAZ - 40' неопасные

    3. По габаритам

    • 20OOG - 20' негабаритные
    • 20STD - 20' стандартные
    • 40OOG - 40' негабаритные
    • 40STD - 40' стандартные

    4. По температурному режиму

    • 20REEF - 20' рефрижераторные
    • 20DRY - 20' сухие
    • 40REEF - 40' рефрижераторные
    • 40DRY - 40' сухие

    5. По состоянию

    • 20DAM - 20' поврежденные
    • 20OK - 20' исправные
    • 40DAM - 40' поврежденные
    • 40OK - 40' исправные

    Рекомендации по применению:

    1. Комбинируйте фильтры для точного распределения (например: 40FULL_HAZ)
    2. Для негабаритных грузов добавьте отдельные правила обработки
    3. Поврежденные контейнеры направляйте в специальные зоны ремонта
    4. Рефрижераторные контейнеры привязывайте к оборудованию с питанием

    Важно: Обновляйте фильтры при появлении новых типов контейнеров или изменении требований безопасности.
